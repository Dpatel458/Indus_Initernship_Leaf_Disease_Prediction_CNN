ğŸ“ Model Metrics

ğŸ”¸ Purpose:

This section presents key evaluation metrics that reflect the performance of the machine learning model used for predicting plant diseases. It helps users understand how accurate and reliable the model is across different classes.

ğŸ”¸ Key Features:

  • Accuracy: Overall percentage of correctly predicted labels.
  • Precision: Measures the correctness of positive predictions (per class).
  • Recall (Sensitivity): Measures how well the model identifies all relevant instances.
  • F1-Score: Harmonic mean of precision and recall, providing a balance between them.
  • Confusion Matrix: A visual grid showing true vs. predicted labels for all classes.
  • Class-wise Analysis: Evaluates how the model performs on each specific crop or disease.

ğŸ”¸ Design Suggestions:

  • Use colorful bar charts or tables to display metrics clearly.

  • Use tooltips or labels to explain each metric for non-technical users.
  • Group metrics by class or show macro/micro averages.

ğŸ”¸ User Benefit:

  • Helps users evaluate the trustworthiness of predictions.
